# API Visualisierung

In diesem Miniprojekt wirst du ein webbasiertes, grafisches Interface für eine REST-API entwickeln. Suche dir dafür aus der folgenden Liste eine API aus, die dich interessiert.

- [Rick & Morty](https://rickandmortyapi.com/)
- [Pokemon](https://pokeapi.co/)
- [Data USA](https://datausa.io/)
- [Weather](https://www.weatherbit.io/)
- [Star Wars](https://swapi.dev/)
- [Cryptowährungen](https://docs.coincap.io/)
- Eine andere, die du im Internet gefunden hast. Kläre diese jedoch mit der betreuenden Person ab.

Identifiziere die Entitäten der API. Zum Beispiel Charaktere, Orte, Währungen. Entwickle auf deiner Webseite folgendes:

1. Eine Hauptseite, in welcher alle Entitäten kompakt und übersichtlich dargestellt sind.
2. Jede Entität soll clickbar sein. Bei einem Click werden weitere, detailliertere Informationen zu dieser Entität angezeigt.

## Bonus

Wenn du mit der Hauptaufgabe bereits fertig bist, kannst du auf der Hauptseite einen Filter einbauen, der die Einträge nach sinnvollen Attributen sortieren kann. Weiter kannst du eine Textsuche einbauen.
